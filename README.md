# apikataromannumber

kata pour faire un convertisseur de nombre ( exemple : 10, 1101) en nombre romain ( exemple : X, MCI)

1 -> I

2 -> II

3 -> III

4 -> IV

5 -> V

6 -> VI

7 -> VII

8 -> VIII

9 -> IX

10 -> X

11 -> XI

...

50 -> L

100 -> C

500 -> D

1000 -> M

2020 -> MMXX

2021 -> MMXXI
